(1) 사진 관련 기능정의

* 사진 올리기
	- 기본 사항
		-> 로그인한 이용자만 사진 올리기 가능
		-> 사진은 이용자의 개인 공간에 올림
	- 필터 적용
		-> 미리 만들어진 필터 제공
		-> 이용자는 필터는 선택해서 사진을 보정
		-> 사진을 올릴 때에만 보정가능하며, 이미 올라간 사진은 보정 불가능
	- 미리보기 이미지(thumbnail image)
		-> 최종 사진 업로드 시 미리보기 이미지 생성
	- 사진 자르기, 크기 조절
		-> 640x640 정사각형 사진 업로드
		-> 크기가 큰 사진인 경우 가로 또는 세로를 640으로 축소 후 정사각형으로 잘라내기함
		-> 640보다 작은 사진인 경우 등록 불가
* 사진 보기
	- 기본 사항
		-> 목록 형식으로 나열하거나 개별 사진을 따로 보기 가능
		-> 모바일에서는 목록으로 보든 개별로 보든 동일한 디자인 적용
		-> 웹에 개별 사진 단위로 게재할 수 있으며, 별도 화면 디자인 적용
		-> 웹에서는 미리보기를 제외한 최종 사진을 여러 개 보지 못 하며, 개별 사진만 가능
		-> 항목 : 최종 결과물 사진, 올린 사람, 작성일시, 사진에 놓은 본문
	- 사진 모아보기
		-> 목록 형식으로 표시
		-> 인기사진
		-> 이용자 개별 타임라인
		-> 특정 이용가자 올린 사진
	- 좋아요 표시 남기기
		-> 좋아요 버튼, 좋아요 표시 남긴 사람
	- 댓글 남기기, 지우기
		-> 댓글 작성자, 본문, 작성일시
		-> 댓글 삭제, 사용자 지정해서 댓글 달기
* 사진 삭제
	- 개별 사진 영역에서 사진 삭제 버튼 클릭 시 확인 메시지 후 동작
	- 삭제 시 댓글, 태크, 좋아요 등 활동 내역 모두 삭제
* 사람 태그 달기



(2) 이용자 관련 기능 정의
* 회원 가입과 탈퇴
	- 회원 가입
		-> ID, 비밀 번호 입력 받아 직접 가입하는 방식
		-> 페이스북 연결하여 가입하는 기능 (추가 구현 필요)
		-> 정보 : 사용할 ID, 비밀번호, 이메일 주소
	- 캡챠(captcha) 기능 구현 필요 (bot으로 계정 임의 다량 생성 방지 목적)
		-> 이메일주소가 실제 존재하는지 확인
		-> 확인 메일을 이메일로 발송
		-> 확인 받지 못 하면 로그인만 가능하며 사진 관련 기능의 이용 제한
* 로그인, 로그아웃
	- 로그인
		-> ID, 비밀번호를 입력받음
		-> 유효 ID 검사 및 로그인 진행
	- 로그아웃
		-> 로그인 된 상태에서만 로그아웃 가능
* 비밀번호 찾기
	- 이메일 주소 확인 기능을 응용 (추가 구현 필요0
* 프로필 보기
	- 간단한 소개 
		->이용자 프로필 사진, 이용자 ID, 팔로잉(following) 수, 팔로워(follower) 수
	- 이용자의 사진
		-> (모바일) 시간 역순으로 나열. (무한 스크롤 방식)
		-> (웹) 이용자가 올린 사진 중 하나를 무작위 선택 표시
	- 팔로잉 표시
		-> 이용자가 구독하는 팔로잉 수
		-> 프로필 내 팔로잉 수 클릭 시 팔로잉 페이지로 이동
		-> 팔로잉 이용자의 프로필 사진, ID 목록 표시, 프로필 페이지 이동 표시
	- 팔로월 표시
		-> 이용자를 팔로잉하는 팔로워 수
		-> 프로필 내 팔로워 수 클릭 시 팔로워 페이지로 이동
		-> 팔로워 이용자의 프로필 사진, ID 목록 표시, 프로필 페이지 이동 표시
* 구독하기 (following) 기능
	- 팔로잉 등록 및 해제
		-> 구독 상태가 아닌 경우 팔로잉으로 표시
		-> 구독 상태인 경우 팔로잉 해제 



(3) 그외 부가적으로 필요한 기능
* 로그아웃 시 첫 화면
* 로그인 실패 화면
* 404 페이지 발생 시 안내 페이지
* 접근 제한 오류에 대한 처리
* 그 외 오류 발생 시 안내 페이지
* 관리자 페이지
* 존재하는 ID 또는 이메일 주소인지 검사하는 기능

(5) URL 정의
* 개별 사진 보는 것 : /photo/<사진 ID>/
* (모바일)인기사진 목록 : /popular/
* (모바일)이용자 개별 타임라인 : /timeline/
* (모바일)특정 이용자가 올린 사진 : /user/<이용자 ID>/
* 좋아요 행위 : /photo/<사진 ID>/like/
* 댓글등록 : /photo/<사진 ID>/comment/
* 댓글지우기 : /photo/<사진 ID>/comment/<댓글 ID>/delete/
* 댓글 가져오기 : /photo/<사진 ID>/get_commens/
* 태그 등록 : /photo/<사진 ID>/tag/
* 사진 삭제 : /photo/<사진 ID>/delete
* 회원 가입 : /account/registration/
* 로그인 : /accounts/login/
* 로그아웃 : /accounts/logout/
* 프로필 보기 : /user/<이용자 ID>/
* 팔로잉 페이지 : /user/<이용자 ID>/following/
* 팔로워 페이지 : /user/<이용자 ID>/followers/
* 구독하기 : /user/<이용자 ID>/follow/
